## Aim:
To write a python program to find the inverse of a matrix

## Algorithm:
### Step1 : Import the NumPy library and define the matrix using np.array().
### Step 2: Ensure the matrix is a square matrix (rows = columns) and its determinant ≠ 0.
### Step 3: Use the NumPy function np.linalg.inv(matrix) to compute the inverse.
### Step 4: Print the inverse matrix using the print() function.

## Program:
```

import numpy as np
matrixA=np.array([[6,2,3],[3,1,1],[10,3,4]])
inverse=np.linalg.inv(matrixA)
print(inverse)

```
